# StockMaster - Portal Web

## 🏢 Qué empresa representa la web

StockMaster es una empresa ficticia dedicada a la **gestión de fondos de inversión**. Su actividad principal consiste en administrar el capital de clientes, invirtiéndolo en diferentes fondos compuestos por activos financieros como acciones, bonos y otros instrumentos del mercado.

La plataforma simula el funcionamiento de una gestora moderna, ofreciendo información sobre inversión, fondos disponibles y servicios de asesoramiento financiero.

---

## 🌐 Qué páginas incluye

La web está compuesta por las siguientes páginas:

- **Página principal (Home)**: presentación de la empresa, servicios y fondos destacados.
- **Fondos**: listado y descripción de los fondos de inversión disponibles.
- **Contacto**: formulario para solicitar información o asesoramiento.
- **Sobre nosotros**: información sobre la empresa, su filosofía y equipo.
- **Tarifas**: información sobre costes, comisiones y condiciones del servicio.

---

## ⚙️ Características del proyecto

- Uso de **Tailwind CSS** para el diseño y estilos, permitiendo una interfaz moderna, responsive y consistente.
- Implementación de **componentes dinámicos en JavaScript**, especialmente el **header y el footer**, que se cargan de forma automática en todas las páginas para evitar duplicación de código y mantener una estructura más escalable.
- Navegación entre páginas mediante enlaces HTML tradicionales.
- Estructura modular y organizada para facilitar el mantenimiento del proyecto.
