# 📊 Stockmaster – Plataforma de Gestión de Fondos de Inversión

## 🧠 ¿Qué es este proyecto?

Stockmaster es una plataforma completa de gestión de fondos de inversión que simula el funcionamiento real de una empresa financiera.

El sistema permite gestionar clientes, fondos, activos financieros, transacciones y datos de mercado, integrando tanto una base de datos relacional como una aplicación web y lógica de backend.

---

## 🎯 ¿Qué problema resuelve?

Este proyecto tiene como objetivo simular la gestión real de una empresa de inversión, resolviendo problemas como:

- Gestión de carteras de clientes
- Control de fondos de inversión
- Registro de transacciones financieras
- Seguimiento de activos y precios de mercado
- Análisis de posiciones y rendimiento

Permite centralizar toda la información financiera en un sistema estructurado, consistente y escalable.

---

## 🛠️ Tecnologías utilizadas

- Java (Lógica de negocio)
- JDBC (Acceso a datos)
- MySQL (Base de datos relacional)
- HTML5 / Tailwind (Interfaz web)
- JavaScript (Navegacion web)
- DBeaver (Gestión de base de datos)
- Git & GitHub (Control de versiones)

---

## 🚀 Instalación y ejecución

- Para ejecutar el panel de inversion - mvn javafx:run
- Para ejecutar la web con vite - npm run dev
